# Changelog

## FBIde 0.5.0.beta-1 — 2026-05-03

First public pre-release of the FBIde rewrite. The codebase is now a clean
modern C++23 project built on wxWidgets 3.3, with the original FBIde
feature set restored on top.

- Editor: syntax highlighting, code folding, auto-indent, context-sensitive help.
- Compiler integration: compile, run, quick-run, parameters dialog, command prompt.
- Tooling: code formatter (case conversion + reformat), Sub / Function browser,
  multi-line find / replace, recent files, multi-document tabs.
- Themes: Borland, Classic, Matrix, Modern (light + dark), Obsidian, Ocean, Twilight.
- Localisation: English plus a dozen language translations.
- Single-instance handling, drag-and-drop file open, command-line arguments.
